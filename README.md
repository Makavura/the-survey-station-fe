# Survey Front End
*Built on Angular 12*

## Running the code
- git clone git@github.com:Makavura/the-survey-station-fe.git
- cd the-survey-station-fe
- npm i or yarn
- ng serve

## Features
- Create Survey with Questions
- View Survey Responses
- View Survey Reports
